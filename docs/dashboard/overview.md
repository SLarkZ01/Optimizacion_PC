# Dashboard Overview

- Ultima actualizacion: Marzo 2026
- Owner: Admin Panel

Panel de administracion protegido para gestionar clientes, compras, reservas y precios.

Se divide en rutas de resumen, tablas paginadas y una seccion de configuracion de precios.

## Enfoque de metricas financieras

- El KPI principal de ingresos usa monto neto (lo realmente recibido).
- Se muestra contexto de bruto y comision en tablas y detalles para trazabilidad.
