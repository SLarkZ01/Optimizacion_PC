# Estructura de Carpetas

- Ultima actualizacion: Marzo 2026
- Owner: Engineering

## Directorios principales

- `app/`: rutas, layouts, API handlers
- `components/`: UI compartida, landing y dashboard
- `hooks/`: hooks reutilizables
- `lib/`: utilidades, constantes, tipos e integraciones
- `docs/`: documentacion tecnica
- `supabase/`: schema SQL

## Archivos clave

- `proxy.ts`: proteccion de rutas privadas del dashboard
- `vitest.config.ts`: config de tests con Vitest
- `vitest.setup.ts`: setup de jest-dom para tests
